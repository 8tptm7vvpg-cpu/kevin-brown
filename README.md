# Kevin Brown Creative — Wix landing page concept

## Recommended Wix setup

The simplest and most reliable approach is to host this folder as a small standalone page, then use Wix's **Embed a Site** element to place its URL on a new Wix page. This avoids Wix iframe code-length limits and preserves the responsive CSS and animation.

### Option A — GitHub Pages (recommended)
1. Create a new GitHub repository.
2. Upload `index.html` and the `assets` folder without changing the structure.
3. Open **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**, select `main` and `/root`, then save.
5. Copy the live GitHub Pages URL.
6. In Wix, create a blank page.
7. Choose **Add Elements → Embed Code → Embed a Site**.
8. Paste the GitHub Pages URL.
9. Stretch the embed to the full page width and make it tall enough to prevent an inner scrollbar.

### Option B — Paste raw HTML into Wix
Wix also offers **Embed HTML / Embed Code**, but a long page can be harder to maintain there. Use this only for a smaller section, not the entire page.
